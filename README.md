技术栈：NodeJs + Typescript + Fastify


## 如何进行二次开发

使用`nodemon`，直接输入`npm run dev`运行即可，修改代码后可直接完成启动部署。

## 如何启动

```shell
npm start
```

项目默认运行至3000端口，使用宝塔面板/Nginx+ACME反向代理体验更佳。

# 特点

自动对接微信后端，无需集成至已有后端项目。
